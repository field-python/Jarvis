# PeePee Benchmark Scorecard

Use this file to track how well PeePee answers important questions over time.

## Rating Scale

- 5 = excellent, accurate, practical, well-scoped, and useful immediately
- 4 = good, mostly correct, minor gaps
- 3 = mixed, partially useful but incomplete or noisy
- 2 = weak, major gaps or poor retrieval
- 1 = bad, misleading, invented, or not useful

## What To Watch For

- Does it answer the actual question?
- Does it avoid inventing fake files or fake source packages?
- Does it use Alaska-specific context when relevant?
- Does it avoid over-citing unless asked?
- Are the "Next questions" actually helpful?
- Does it rely on real archive evidence instead of templates or metadata?
